Spor bøkene dine med Plots, den ultimate bok tracker og leselog app. Registrer enkelt bøker, filmer og serier som notater, vurder med 5 stjerner og organiser i mapper. Synkroniser sikkert via iCloud.
Administrer lesejournalen og filmloggen uten anstrengelse.

Hvorfor Plots skiller seg ut:
• Dype notater for sider, sitater og tanker
• Intuitiv 5-stjernet vurdering
• Kategoriser etter sjanger eller serie
• Stjernebildetagger for unike vibber
• Minimalistisk UI enkel å bruke
• Automatisk iCloud backup

Bygg lesevaner og inspirer til neste. Last ned Plots nå og begynn å logge reisen din!
