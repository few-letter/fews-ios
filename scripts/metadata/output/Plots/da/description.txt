Spor dine bøger med Plots, den ultimative bog tracker og læselog app. Registrer let bøger, film og serier som noter, bedøm med 5 stjerner og organiser i mapper. Synkroniser sikkert via iCloud.
Administrer din læsejournal og filmlog uden besvær.

Hvorfor Plots skiller sig ud:
• Dybe noter til sider, citater og tanker
• Intuitiv 5-stjernet bedømmelse
• Kategoriser efter genre eller serie
• Stjernebilled tags til unikke vibes
• Minimalistisk UI let at bruge
• Automatisk iCloud backup
